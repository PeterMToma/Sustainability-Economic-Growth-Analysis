# Sustainability-Economic-Growth-Analysis
This project analyzes the relationship between sustainability and economic growth across selected countries from 2010 t0 2024. It develops a composite Sustainability Index (CSI) based on environmental, economic, social, energy transition, and smart sustainability dimensions to compare countries and assess sustainable development progress over time.

# Project Period
2010-2024

- PM2.5 Air Pollution
- Renewable Energy Consumption

- Waste Management / Recycling Rate
  
  # Economic Index

- Inflation Rate
  
  # Social Index

- Poverty Rate
- Access to Clean Water
  
  # Energy & Transition Index

- Fossil Fuel Consumption
- Electricity Access

  # Smart Sustainability Index
- Smart Cities Development Index
- Rural Digitalization Index
- Digital Infrastructure for Sustainability
